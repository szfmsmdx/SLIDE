# 手写神经网络...
> 本项目为笔者毕设的一部分工作内容  
> 主要是基于 python 从零开始手写一个神经网络 hh  
> 之后考虑多线程性能可能会转 C++ 翻译一遍。。。


## 基本思路
利用 LSH 优化神经网络中对神经元的选择更新，以加速神经网络的训练

考虑到，利用已经封装好的框架，首先不方便插入LSH，其次，其封装的效果可能比使用LSH的效果来的更好

因此，笔者决定只使用 numpy 手写一个简单的神经网络，然后比较不同规模、不同任务下 LSH 加入与否的效果
